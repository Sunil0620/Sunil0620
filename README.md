## 🎯 Featured Engineering Projects

### 🌌 [Orbit](https://github.com/Sunil0620/orbit) — High-Throughput Real-Time Collaboration Hub
- **The Problem:** Traditional chat clients experience high latency, packet drop, and poor state synchronization when handling sudden connection fluctuations.
- **The Solution:** Built a real-time team messaging service leveraging Django Channels (WebSockets) for long-lived socket management, React on the client-side for dynamic view synchronization, and Redis as an in-memory message broker to handle pub/sub routing.
- **Impact:** Achieved sub-100ms real-time delivery and seamless socket recovery protocols on connection dropouts.
- **Tech Stack:** `Django Channels` `Redis` `React` `PostgreSQL` `WebSockets`

### 📝 [Django Blog](https://github.com/Sunil0620/django-blog) — Modular Content Delivery Platform
- **The Problem:** Modern content systems are often bloated with third-party tracking, slow databases, and complex authorization mechanisms.
- **The Solution:** Engineered a clean, secure blogging platform featuring custom markdown integration, optimized database querying (Django ORM optimization), and a robust role-based user authentication model.
- **Impact:** Light-weight content delivery with rapid response times and SEO-optimized dynamic slug routing.
- **Tech Stack:** `Python` `Django` `Bootstrap` `SQLite` `Authentication API`

### 🎨 [Portfolio](https://github.com/Sunil0620/Portfolio) — High-Performance Interaction Showcase
- **The Problem:** Portfolios are frequently heavy, loading slowly due to bloated frameworks, and missing clean semantic access.
- **The Solution:** Created a responsive, lightweight developer portfolio site written with semantic HTML5 and vanilla CSS/JS transitions to optimize delivery speed.
- **Impact:** Under 200ms initial load time with a clean 100/100 Lighthouse performance grade.
- **Tech Stack:** `HTML5` `CSS3` `Vanilla JS`
